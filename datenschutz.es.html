2. Alojamiento (GitHub Pages)

Este sitio web está alojado en los servidores de GitHub Pages (GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, EE. UU.). Al acceder al sitio web, GitHub registra datos técnicamente necesarios en archivos de registro del servidor (en particular dirección IP, fecha y hora de la solicitud, página consultada, URL de referencia, navegador y sistema operativo). La base jurídica es el art. 6 apdo. 1 lit. f RGPD (interés legítimo en el funcionamiento técnicamente correcto del sitio web). Para las transferencias de datos a EE. UU., GitHub se basa en el Marco de Privacidad de Datos UE-EE. UU.

Más información: GitHub Privacy Statement.

3. Análisis web con Matomo

Este sitio web utiliza Matomo (anteriormente Piwik), un software de código abierto para el análisis estadístico de las visitas al sitio web. Matomo se ejecuta en un servidor propio en Alemania (Strato AG, Berlín). Sus datos no se transmiten a terceros.

Durante el análisis se procesan los siguientes datos: la dirección IP anonimizada, las páginas consultadas, la fecha y hora del acceso, el tipo de navegador y sistema operativo, así como la página de origen (referrer). La dirección IP se anonimiza antes de almacenarse, de modo que no es posible inferir información sobre usuarios individuales.

Base jurídica: art. 6 apdo. 1 lit. f RGPD (interés legítimo en el análisis anonimizado del comportamiento de los usuarios para mejorar la oferta web).
